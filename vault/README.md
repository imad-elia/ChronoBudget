# ChronoBudget — Knowledge Vault

Obsidian vault for persistent memory across Claude Code sessions.

## Structure

| Folder | Contents |
|--------|----------|
| `Projects/` | Project overview, stack, status |
| `Architecture/` | System design, components, data flow, schema |
| `Decisions/` | Architectural decision records (ADRs) |
| `Sessions/` | Per-session notes |
| `Issues/` | Open bugs and known gaps |
| `Templates/` | Blank templates for new notes |
